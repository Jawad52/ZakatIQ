 "ZakatIQ" — AI-Powered Zakat & Halal Finance Assistant

The Core Idea
A web/mobile app that helps Islamic bank customers:

Calculate their Zakat automatically by connecting to their account balances
Get AI guidance on whether their investments/savings are Sharia-compliant
Distribute Zakat directly to verified causes through the bank

📱 Key Features (Keep it to 3 for the competition)
Feature 1 — Smart Zakat Calculator

User inputs or simulates their asset types (savings, gold, investments)
App calculates Nisab threshold and exact Zakat amount
Clean, animated UI showing the breakdown

Feature 2 — AI Sharia Compliance Checker

User types a financial question (e.g. "Is investing in this ETF halal?")
Powered by Claude API (vibe coded — you just write the prompt)
Returns a clear Yes / No / Needs Review with reasoning

Feature 3 — Zakat Distribution Hub

Shows verified charitable causes (simulated for the competition)
One-tap "Donate via [Your Bank Name]" button
Gives a sense of full end-to-end journey
